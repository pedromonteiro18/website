Projects

Rover
During my first year at univeristy, I was part of a project in which we created a remote controlled rover capable of identifying different minerals using infrared, radio, ultrasonic and magnetic sensors. These sensors were used to identify certain properties of the minerals and searched up on a database containing all possible minerals and their properties to identify the correct rock. I was responsible for programming the different sensors and the detection software to analyse the data and output the correct mineral.

Smart Alarm
During my second year, I was involved with a project to create a smart alarm system for bikes. The alarm would be activated and deactivated using a mobile application and would also detect certain patterns and motions to determine if the bike is being handled in a which seems that it is likely being tampered with. To do this, we used an accelermometer and a gyroscope. I was tasked with designing and programming the mobile application and the microcontroller, whilst also helping build the device.
